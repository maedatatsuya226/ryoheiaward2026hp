# 良平アワード2026 公式Webサイト 実装指示書

## 1. プロジェクト概要

「良平アワード2026」の公式Webサイトを制作する。

本サイトは、故・蒲池良平先生の精神と医療への姿勢を継承し、医療現場で日々積み重ねられている誠実な行いや、周囲を支える行動に光を当てるための公開サイトである。

審査員向けページ、事前審査、評価、集計機能は別システムで運用しているため、本サイトには含めない。

本サイトは、一般参加者・関係者向けに以下を伝えることを目的とする。

- 良平アワードの趣旨
- 蒲池良平先生の志
- 良平アワード2026の開催情報
- 過去の授賞式の歩み
- 受賞者の行動や思い
- 参加者一人ひとりが志の継承者であること

---

### 7.4 良平アワードが光を当てる行動

- 誰かのために手を伸ばす

### 7.5 過去の授賞式・これまでの歩み

過去の授賞式写真を横にスライドして表示する。

単なる画像カルーセルではなく、

> 受け継がれてきた時間をたどる写真回廊

として見せる。

見出し候補：

- これまでの歩み
- 受け継がれてきた時間
- 志がつながる瞬間

実装要件：

- 6〜10枚程度を想定
- 1枚を大きめに表示
- 左右の写真が少し見切れる
- 年度と短いキャプションを表示できる
- PCは横スクロールまたはゆっくり自動再生
- 自動再生する場合は速度を遅くする
- ホバー、フォーカス、タップで停止できる
- スマートフォンではスワイプ操作に対応
- キーボード操作にも対応
- `prefers-reduced-motion` に対応
- 画像の追加・差し替えをデータ配列から行えるようにする

仮キャプション例：

- 2023年 第1回 良平アワード
- 想いをつなぐ最初の一歩
- 仲間への敬意があふれた時間
- 笑顔と拍手に包まれた授賞式
- 志が次の世代へ受け継がれる瞬間

仮画像であることが分かるよう、ダミーの人物写真は使用しないこと。

- 上品さ：6
- ワクワク感：4

故人を偲ぶ静けさや重みは持たせるが、暗く悲しい印象には寄せない。

前向きで温かく、これから始まる特別な時間への期待感を感じるデザインとする。

- 敬意
- 日々の努力
- 光
- 温かさ
- 高揚感
- 期待感
- 静かな感動
- 未来へ続く志

```css
:root {
```

## 14. アニメーション方針

上品で遅めの動きを使用する。

使用候補：

- フェード
- 軽いスライドアップ
- 写真のゆっくりした横移動
- 光のごく弱い変化
- スクロールに応じた自然な表示

必須要件：

- `prefers-reduced-motion` に対応
- アニメーションなしでも内容が理解できる
- 操作を妨げない
- モバイルで負荷をかけすぎない

## 16. コンテンツ管理

後から非エンジニアでも差し替えやすいよう、文章や開催情報をデータファイルに分離する。

例：

```ts
export const eventInfo = {
  title: "良平アワード2026",
  venue: "令和健康科学大学",
  date: null,
  time: null,
  organizer: null,
  detailsPublished: false,
}
```

写真回廊：

```ts
export const historyItems = [
  {
    year: "2023",
    title: "第1回 良平アワード",
    caption: "想いをつなぐ最初の一歩",
    image: "/images/history/history-01.webp",
  },
]
```

架空データを大量に作成しないこと。

## 24. 完成イメージ

サイトを開くと、深いネイビーと柔らかな光の中に「志を受け継ぎ、日々の行いに光を。」という言葉が現れる。

スクロールすると、良平アワードの趣旨、蒲池良平先生の人柄、医療現場で光を当てたい日々の行動が、写真と余白を使って丁寧に伝わる。

過去の授賞式写真は、横にゆっくりと流れる写真回廊として表示され、アワードが積み重ねてきた時間を感じられる。

後半では、良平アワード2026の開催概要と会場情報を案内し、表彰後は受賞者ページへ自然に誘導する。

最後は、

> 今日ここにいる全員が、その志の継承者です。

というメッセージで締めくくる。

全体として、追悼だけに寄りすぎず、品格、温かさ、期待感、未来への継承を感じるWebサイトとする。
